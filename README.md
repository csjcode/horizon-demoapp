# horizon-demoapp

### Initial install

* >npm install -g horizon
* >hz init example_app
* Test server out with: hz serve --dev
* http://127.0.0.1:8181/

### Chat app example

* app.js with the chat logic.
* index.html with horzon library and vue.js cdn
* hz serve --dev
* http://127.0.0.1:8181/
* Note that input with enter updates in multiple tabs
